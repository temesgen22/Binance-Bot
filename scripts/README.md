# Scripts Directory

This directory contains utility scripts for development, testing, and maintenance.

## Available Scripts

### `check_redis.py`
Check Redis connection and configuration.

### `diagnose_redis_issue.py`
Diagnose Redis connection issues.

### `test_redis_integration.py`
Test Redis integration functionality.

### `test_redis_warning_fix.py`
Test Redis warning fixes.

### `view_candles_example.py`
Example script to view Binance candlestick data.

## Usage

Run scripts from the project root:

```bash
python scripts/view_candles_example.py BTCUSDT 1m
```
